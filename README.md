## ✨ Features

- 📰 **Editorial Aesthetics**: Designed like a vintage final-edition newspaper dispatch with modern interactivity.
- 🎛️ **GSAP Horizontal Spread**: A pinned sideways-sliding project gallery that responds cleanly to scrolling.
- 🎨 **Dynamic Color-Shifting**: Ambient theme shifting that follows the viewport section seamlessly.
- 🌊 **Interactive Text Ripples**: Custom letter-ripping physics that react dynamically to cursor movements.
- 💨 **Ambient Dust & Grain**: Fine-tuned particle effects and CSS grain for texture.
- ⌨️ **Press [G] for Chaos**: Hidden interactive mode.

## 🛠️ Tech Stack

- **Structure & Logic**: HTML5 + Vanilla JS (ES6+)
- **Animations**: GSAP (GreenSock) + ScrollTrigger
- **Styling**: Pure CSS (Variables, clamp sizing, CSS Grid/Flexbox)
- **Effects**: HTML5 Canvas (Particle simulation)
